## Descrição do caso escolhido 
O projeto Lucas.Dev é um projeto de landing page desenvolvida para apresentar os serviços de um desenvolvedor freelancer e facilitar o contato com novos possíveis clientes interessados em desenvolver sistemas, aplicações web, aplicativos móveis ou APIs.

A página foi criada com foco em transmitir profissionalismo, apresentar de forma clara os serviços oferecidos e incentivar o visitante a solicitar um orçamento por meio do formulário de contato.

## Necessidade identificada 
Muitos profissionais autônomos da área de tecnologia possuem conhecimento técnico, mas não dispõem de uma página própria para apresentar seus serviços de forma organizada e profissional.

Pensando nisso, desenvolvemos uma landing page capaz de reunir as principais informações sobre o desenvolvedor, destacando seus serviços, diferenciais e disponibilizando um canal direto para contato com novos potenciais clientes.

## Público-alvo 
- Pequenos empresários
- Empreendedores
- Pessoas que desejam transformar uma ideia em software.
- Empresas que precisam desenvolver sistemas próprios.
## Objetivo do site 
O principal objetivo do site é apresentar os serviços do desenvolvedor de forma clara e profissional, demonstrando como a tecnologia pode solucionar problemas reais.

Ao final da navegação, espera-se que o visitante preencha o formulário para entrar em contato com o desenvolvedor e solicitar um orçamento para seu projeto.

## Processo de desenvolvimento 
O desenvolvimento começou com a ideia de criar um portfólio tradicional. No entanto, durante o planejamento, percebemos que uma landing page voltada para conversão de clientes seria mais eficiente.

O nosso maior desafio não foi a construção do HTML, mas sim selecionar apenas as informações realmente relevantes para um potencial cliente. O foco passou a ser uma navegação simples, objetiva e que conduzisse naturalmente o visitante até o formulário de contato.

Também foi dada atenção especial à organização visual, utilizando uma identidade moderna baseada em tons escuros com detalhes em azul para transmitir tecnologia e profissionalismo.

## Principais decisões do projeto 
- Utilização de HTML semântico para facilitar a organização do código;
- Layout responsivo utilizando Bootstrap 5;
- Identidade visual minimalista, com cores escuras e detalhes em azul;
- Navegação simples e objetiva;
- Uso de ícones do Bootstrap Icons para melhorar a experiência visual;
- Menu responsivo (hamburger) para dispositivos móveis;
- Formulário de contato intuitivo e de fácil preenchimento;

## Funcionalidade em JavaScript 
Foi implementada uma simulação de envio do formulário de contato.

Como o projeto não possui integração com um backend, o JavaScript intercepta o envio do formulário, exibe uma mensagem de sucesso ao usuário e limpa automaticamente todos os campos preenchidos, simulando o comportamento de um envio real.

Também foi desenvolvido um menu responsivo (hamburger), permitindo uma boa navegação em dispositivos móveis.

## Uso de Bootstrap 
Foi utilizado no projeto, classes simples e o uso de container nas seções do projeto.
Como na linha 16 do HTML:
```html
<div class="container d-flex justify-content-between align-items-center">
```
Dessa forma, não há necessidade de criar uma classe no css para estilizar esse bloco, basta usar as tags fornecidas pelo bootstrap.
